# 🦄 Unicorn Hunter (유니콘 헌터)

**"당신에게 1,000만 원이 있다면, 과거의 이 스타트업에 투자하시겠습니까?"**

Unicorn Hunter는 실제 스타트업의 초기 아이디어(Blind Pitch)만을 보고 투자를 결정하는 **투자 시뮬레이션 게임**입니다.
성공한 유니콘 기업의 반전 스토리와, 실패한 기업의 치명적인 실수를 통해 비즈니스 인사이트를 얻어보세요.

## 🎮 게임 방법
1. **자본금 1,000만 원**으로 시작합니다.
2. 매 라운드마다 익명의 스타트업 8곳이 랜덤으로 등장합니다.
3. 기업의 **초기 아이디어**와 당시 **투자자들의 반응**을 읽습니다.
4. `PASS`, `100만 원`, `500만 원`, `ALL-IN` 중 하나를 선택해 베팅합니다.
5. 결과를 확인하고, 자산을 불려 **'투자의 신'** 랭킹에 도전하세요!

## 🏢 포함된 기업 케이스 (일부)
총 16개의 전설적인 기업들의 스토리가 수록되어 있습니다.
- **Airbnb:** "누가 남의 집 거실에서 잠을 자?"
- **Uber:** "불법 택시 영업 아니야?"
- **Netflix:** "비디오 대여점이 있는데 굳이 우편으로?"
- **Theranos:** "피 한 방울로 모든 질병을 진단한다고?"
- ...그리고 더 많은 성공과 실패 사례들!
